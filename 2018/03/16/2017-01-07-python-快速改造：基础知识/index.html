三、算法是什么

在开始变成前，首先先解释一下什么是计算机程序设计。简单来说，就是告诉计算机去做什么。计算机可以做很多事情，但是不会自己去做，程序员就像给小孩喂饭一样告诉它具体的细节，并且使用计算机能够理解的语言—-算法。“算法”不过是“步骤”或者“食谱”的另外一种文绉绉的说法—对于如何做某一件事的一份详细描述。比如：
凉皮做法：
（1）将凉皮切成细条
（2）将凉皮放入容器中
（3）放入麻将、醋、蒜汁、辣椒等调料
（4）搅拌大约1分钟左右
其实上面的步骤就是算法，他告诉醋程序如何去做，做多久，以及做什么，还有对什么做。
